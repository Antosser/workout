# [Workout](https://antosser.github.io/workout/)

This is a simple minimalistic open-source web application to start working out

## Usage

1. Go to [this repository's pages](https://antosser.github.io/workout/)
2. Add exercises by clicking the `Add exercise button`
3. Click `Start today's workout` to start working out

### Streak

* Your streak will increment by 1 every day you work out
* Your streak will reset if you forget to exercise one day
